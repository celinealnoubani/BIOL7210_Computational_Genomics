This is the home for MLST final results. MLST was run on all 34 isolates to determine strain type.

# Final Results
All 34 isolates run with MLST produced scores indicating they were strain 10314.

## Command Line Input
```bash
mlst <assembly_directory>/*.fa > mlst_output.tsv
```
